## 初步策略结论候选

一句话：D1 不是越强越好，而是越“干净”越好；真正决定 D3 成败的是 D1 结构干净度 + D2 承接质量 + D3 回收确认。

候选调整方向：

1. 提高 D1 炸板次数惩罚权重。
   - 0 次炸板：优先保留。
   - 1 次炸板：可接受。
   - 2 次炸板：降权。
   - 3 次以上：明显降权。
   - 5 次以上：基本不进核心观察。
   - 极端多炸板：剔除或低频备选。
2. 将 D1 拥挤度纳入更强评分/降权。
   - D1 成交额、换手率不只作为备注，应进入评分体系。
   - 拥挤票不要进入核心观察位，只能备选。
3. D2 的“冲高回落”继续保留，但必须强调回落后不破结构。
   - 冲高回落 + 不破支撑：保留。
   - 高开低走、收盘破位：剔除或降权。
   - D2 量比 2–3 倍时需要更强补强条件。
4. D3 只做最终确认，不承担“救票”功能。
   - 重点看是否回收观察价、是否站稳、是否持续承接。
5. 观察池分层：核心观察 / 强势确认观察 / 低频备选 / 剔除。
