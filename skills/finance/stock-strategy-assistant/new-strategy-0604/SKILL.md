# 屠龙战术

A 股短线五日线战法。核心逻辑：捕捉首板主力试盘后次日洗盘，第三日水下低吸，第四五日快进快出。

## 适用场景

- 用户要求按「屠龙战术」分析某只股票或某个交易日
- 用户要求从昨日首板中筛选符合战法的标的
- 用户询问第三日是否适合买入、第四日如何卖或做 T
- 用户引用天安新材（603725）、佳力图（603912）等案例

## 五日线流程


| 日     | 动作         | 要点                      |
| ----- | ---------- | ----------------------- |
| 第 1 天 | 观察         | 盘中突然强势拉升，收 **首板**       |
| 第 2 天 | 确认         | 冲高回落；**最好低开**，上午冲高、下午回落 |
| 第 3 天 | **买入**     | 必须 **水下** 才买            |
| 第 4 天 | **卖出 / T** | 按开盘形态决策，默认不恋战           |
| 第 5 天 | **清仓**     | 除非大行情，否则全清              |


---

## 第 1 天：首板筛选

在 **通达信** 找出 **昨日首板**，再从中筛选候选。

### 数据来源口径

优先使用用户通达信里的 **“昨日首板”条件池**。如果用户没有提供截图或导出文件，则先由助手自动复刻通达信口径生成 D1 池：

```text
昨日涨停
且前一交易日不是涨停
剔除 ST / *ST / 退市
按 10cm / 20cm / 30cm / 北交所涨跌幅规则分别判断涨停
保留反包首板、断板后再首板等通达信可能纳入的形态
```

公开复盘源只作为补充校验，不作为主池唯一来源。若复刻结果与用户通达信池有差异，以用户通达信结果为准，并用差异继续校准规则。

**通达信筛选路径（参考）**

1. 功能 → 选股器 → 条件选股
2. 添加条件：`首板` 或 `昨日涨停 AND 非连板`（具体公式名因版本而异）
3. 可选叠加：非 ST、流通市值区间、所属板块强度
4. 导出列表，逐只标记第 1 天 K 线是否为 **盘中突然强势拉升**（非一字、非尾盘偷袭）

**第 1 天合格特征**

- 盘中突然强势拉升至涨停（首板）
- 主力有明显主动拉升痕迹，非被动跟风一字

## 第 2 天：形态确认（关键过滤日）

走势要 **D2 日内冲高回落**，最好 **低开**，上午冲高、下午回落。

注意：**冲高回落只看第 2 天当天的分时 / K 线结构**，不要求 D2 收盘价低于 D1 收盘价。D2 收盘仍高于 D1 是正常的强势洗盘形态，例如昊华能源、火炬电子这类走势不能因为“D2 高于 D1”直接淘汰。

### 必须排除（反面教材）

- **高开低走且收盘弱** → 主力开始出货，**直接淘汰**
- 高开后回落但 **收盘仍强、量能可控、曲线/趋势好** → 不直接淘汰，进入候选观察
- 参考反面教材：**天安新材 603725，2023-07-28（周五）** — 主力拉高出货

### 成交量硬规则（可略放宽）


| 情况      | 第 2 天量 / 第 1 天量            |
| ------- | -------------------------- |
| **缩量**  | **< 1 倍 → 直接淘汰**（D2 缩量不考虑） |
| 一般      | **1 ~ 2.5 倍**，量能可控         |
| 曲线/趋势不错 | **2.5 ~ 3 倍** 仍可纳入候选       |
| > 3 倍   | **不安全**，可能主力出货，淘汰          |


成交量不是单独的强指标：**D2 超过 2 倍不直接淘汰**。若分时曲线好、日均线向上、EXPMA 强、收盘承接好，可放宽到 3 倍内。

### 第 2 天 K 线形态偏好

**收盘十字星 / 小实体优先**（洗盘末端信号）：

- **优选**：实体占振幅 ≤15%，且有上下影 → 标准十字星
- **次优**：实体占振幅 ≤25%，涨跌幅 |±2%| 以内 → 准十字 / 小实体
- **可选**：实体占振幅略大但有明显上影，且收盘仍有承接 → 强势回落型
- **减分**：第 2 天仍收大阳线（如 +5% 以上）且没有明显回落 → 洗盘不充分，降低优先级

形态优先级高于量能：量能合格但形态差，不能强行入选。

### 长上影降级规则

D2 上影线过长说明冲高后抛压重，若收盘承接不足，需要降级或淘汰：

- 上影占 D2 振幅过高（如 >60%）且收盘靠近低位 → 强降级
- 上影很长、实体很小，但收盘没有重新拉回均价/关键位 → 降低优先级
- 类似平煤股份（601666）这种“上影很长、D3/D4反馈一般”的形态，不放主优先级；只作观察，甚至不考虑
- 长上影只有在收盘仍有明显承接、板块强、D3 开盘承接好时，才可保留

### 第 2 天形态判定口径

- 低开后冲高回落 → 标准形态
- 平开/小高开后冲高回落 → 可接受
- 高开后冲高回落，只要收盘不弱、量能可控、曲线好 → 可接受
- 不要求 D2 收盘低于 D1；D2 收盘高于 D1 仍可入选
- 重点看 D2 当天是否有“冲高 → 回落 → 承接”的洗盘结构

### 强势承接参考（阳光股份型）

用于辅助识别“不水下但很强”的候选，例如阳光股份（000608）这类。**这是参考加分条件，不是必须类型，也不是主筛选方向。**

满足以下条件时，可提高候选优先级：

- D1 为首板，且 D1 封板质量不差
- D2 不缩量，量比通常在 **1 ~ 2.5 倍**，可略高但不能异常放量
- D2 收盘接近十字 / 小实体，或实体很小但上影明显
- D2 当天有明显冲高回落，且收盘仍有承接，不要求低于 D1
- D2 即使不是低开，只要平开/小高开后有冲高回落，也可入选
- 所属板块当日有强度，或同板块前排有正反馈

这类票的核心不是“水下低吸”，而是 **D2 洗盘后 D3 强势承接**。D3 买点允许平开/小高开，但不能追高过多。

注意：不要为了匹配“阳光股份型”而忽略标准屠龙票。正常筛选仍以 D1 首板、D2 冲高回落承接、量能不缩、无负面消息为主；强势承接只是买点执行和优先级判断的补充。

### 失败屠龙票淘汰模式

后续遇到类似走势要淘汰或强降级：

- **D2 缩量**：D2 量比 < 1，直接淘汰，例如高新发展（000628）失败案例
- **D2 太强但无充分洗盘**：D2 收大阳（如 +5% 以上）、实体偏大、没有明显回落承接，强降级；若 D3 大跌，确认为失败模式，例如光智科技（300489）、恒盛能源（605580）
- **D2 继续强板 / 强势延续**：不直接放入标准屠龙主池，但可以单独找出来分析。此类票可能 D3 低开反包或继续加速，也可能 D4 闷杀，必须标记为高风险观察，并说明买点与兑现纪律，例如金钼股份（601958）、沃格光电（603773）
- **D2 实体过大**：实体占振幅明显偏大（如 >35%），不像十字/小实体洗盘，强降级
- **消息面风险**：即使 D2 形态好，若近期有澄清风险提示、业绩亏损且股价偏离基本面、监管/减持等，淘汰，例如北京科锐（002350）
- **D3 反馈验证失败**：若用于复盘，D3 出现大阴、跌停或明显破位，标记为失败屠龙票；以后识别相同 D1/D2 特征时提前剔除

详细失败案例见 [failure-samples.md](failure-samples.md)。失败样本只说明该票在当时市场、板块、消息面和量价条件下失败，不是永久黑名单；后续重新出现时需按新的环境重新评估。

### 消息面过滤

第 3 日买入前，排查 **近 5–7 个交易日** 是否有重大负面，有则淘汰：

- 股东/高管减持（尤其首板前后）
- 监管处罚、立案、ST 风险
- 业绩爆雷、重大诉讼、补税等
- 行业突发利空直接冲击标的

无上述事项 → 消息面通过。

### 正面教材

**佳力图 603912，2022-12-29（周四）** — 低开、上午冲高、下午回落，量能可控。

详见 [examples.md](examples.md)。

## 第 3 天：买入（优先水下，强势票可例外）

**原则：优先水下买，但不是 100% 硬规则。**

注意：**买点是第 3 天的执行条件，不是选股筛选指标**。筛选候选股时只依据 **D1 首板** 与 **D2 确认**；D3 当天是否给出水下、平开或小高开承接机会，只决定是否出手，不用于淘汰候选。


| 情形   | 操作                                |
| ---- | --------------------------------- |
| 标准洗盘票 | 优先等 **水下** 再买；**上午 10 点左右** 常出现当日低位 |
| 低开很多 | 可 **开盘就买**（仍须满足下方买入校验）            |
| 强势票 | 不一定会水下低开，可在 **平开 / 小高开且承接强** 时买入 |
| 高开过多 | 不追高；若快速冲高无回落承接，放弃 |


### 强势票 D3 买点例外

满足以下条件时，D3 不水下也可买：

- D1/D2 形态强，D2 有冲高回落承接，量能可控
- D3 平开或小高开，开盘后不快速下杀
- 分时回踩不破开盘价或 D2 收盘附近，承接明显
- 板块/题材同步强势，龙头或前排有正反馈
- 买入位置不能追在快速拉升后，只能在回踩承接处介入

强势票若高开过多、直接加速、没有回踩承接，宁可放弃。

### D2 继续强板 / 强势延续票

这类票不是标准屠龙主模型，但可以作为单独观察对象输出：

- D1 首板后，D2 继续大阳、接近涨停或继续涨停
- D2 通常不是十字/小实体洗盘，实体偏大或收盘靠近高位
- D3 可能出现低开反包涨停，也可能高开加速
- D4 风险明显更大，容易冲高回落、闷杀或跌停

### 强势票二进三评估

如果 D2 是继续强板 / 强势延续，需要额外评估是否具备 **二进三** 价值：

- 所属题材是当日主线，板块有 3 只以上涨停或前排持续正反馈
- D2 封板质量好：封板早、炸板少、回封强、封单不弱
- D2 量能不是异常爆量，且换手充分但不失控
- D3 竞价不能明显弱：不能大幅低开无承接，也不能高开过多后快速下杀
- D3 开盘后回踩承接强，分时不破均价线或能快速收回
- 同梯队或更高标没有明显负反馈

满足以上条件，可标记为“二进三观察”；否则只作为强势延续高风险观察，不参与。

二进三纪律：

- 只买分歧后回封、回踩承接或小高开稳住后的确认点
- 不追一字板后排，不追高开秒板后排
- 若 D3 封不住或炸板回落，及时放弃或离场
- D3 成功晋级后，D4 高开/冲高优先兑现，不恋战

输出时必须单独标注：

- 不属于标准屠龙低吸票
- 属于高风险强势延续观察
- D3 只能买低开承接或回踩承接，不能追高
- D3 若大涨/涨停，D4 必须严格兑现；低开弱走要直接卖

### 阳光股份型 D3 执行参考

若 D1/D2 已呈现类似“强势承接”特征：

- D3 平开 / 小高开可买，不要求水下
- 优先买在开盘后第一次回踩承接处
- 若 D3 开盘涨幅过大（如 >5%）且快速拉升无回踩，放弃
- D3 若快速封板，可视为错过，不追板
- D4 高开或冲高按纪律止盈，不恋战


### 第三日是否适合买入 — 五项校验（全部满足才出手）

```
买入校验：
- [ ] 1. 第二天成交量 ≥ 第一天 1 倍（缩量不考虑）；一般 ≤ 2.5 倍，曲线/趋势好最多 3 倍
- [ ] 2. D2 当天有冲高回落结构（只看 D2 日内，不要求 D2 收盘低于 D1）
- [ ] 3. 第 2 天收盘偏十字 / 小实体 / 强势回落承接（优选）
- [ ] 4. 日均线向上
- [ ] 5. EXPMA 在两个趋势线上方
- [ ] 6. DK 点处于 D 状态
- [ ] 7. 近 5–7 日无重大负面消息
- [ ] 8. 有合适机会才出手 — 不要天天都出手
```

**任一不满足 → 放弃本次，不强行交易。**

## 第 4 天：卖出或做 T


| 开盘形态     | 操作      |
| -------- | ------- |
| **高开很多** | 马上卖，不贪心 |
| **平开震荡** | 等待拉高后卖  |
| **低开**   | 见下方分支   |


**低开分支**

- 低位 **不低于** 第 1 日主力拉升 **前** 的位置 → 可根据均线情况考虑 **做 T**
- 低位 **低于** 第 1 日主力拉升前的位置 → **割肉离场，不要做 T**

**仓位**

- 除非遇上 **大行情**，可考虑 **卖半仓**
- 否则 **清仓**

## 第 5 天：清仓

除非遇上 **大行情**，否则 **清全仓**。

## 批量筛选工作流

1. 通达信导出 **昨日首板** 列表
2. 逐只核对 **第 2 天** 形态与量能 → 淘汰不合格
3. 对幸存者做 **第 3 天五项校验**
4. 输出通过 D1/D2 筛选的候选，附第 3 天买入执行计划
5. 强调：第 3 天买点分层执行：标准票优先水下；强势票可接受平开/小高开承接；高开过多不追
6. 强调：**有合适机会才出手，不要天天都出手**

## 术语速查


| 术语         | 含义（本战法内）                   |
| ---------- | -------------------------- |
| 首板         | 当日首次涨停（非连板）                |
| 水下         | 分时 0 轴下方 / 低于关键参考价；第三日不买追高 |
| 第 1 日拉升前位置 | 首板主力启动前的平台或启动低点，作止损与 T 参考  |
| EXPMA      | 指数移动平均线，需在两条趋势线上方          |
| DK 点       | D 状态为买入侧信号（依用户通达信 DK 指标定义） |
| 大行情        | 指数/板块系统性强势，可放宽第 4–5 天持仓比例  |
