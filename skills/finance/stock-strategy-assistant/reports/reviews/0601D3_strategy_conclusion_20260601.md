# 0601D3 策略结论与修订建议（落盘版）

生成时间：2026-06-01

说明：
- 本文基于 0601D3 的 D1 样本与当天 D3 复盘结果，先形成独立结论记录。
- 当前版本仅用于后续验证，不视为最终定稿。

## 核心结论

D1 不是越强越好，而是越“干净”越好；真正决定 D3 成败的，不是首板当天的热闹程度，而是 D1 结构干净度 + D2 承接质量 + D3 回收确认。

## 具体结论

### 1. D1 炸板次数应提高惩罚权重

观察结果：
- 炸板少的首板，后续 D3 成功率更高。
- 炸板很多的首板，即使当天封住，也更容易后面走弱。
- 0601D3 中，最差样本之一甘肃能源就是典型反例。

建议：
- 0 次炸板：优先保留
- 1 次炸板：可接受
- 2 次炸板：降权
- 3 次以上：明显降权
- 5 次以上：基本不进核心观察
- 极端多炸板：建议剔除或只做低频备选

### 2. D1 拥挤度应纳入更强的评分/降权

观察结果：
- D1 成交额过大、换手过高的票，后面更容易出现兑现压力。
- 今天样本里，D3 走得好的票整体比走差组更不拥挤。

建议：
- D1 成交额和换手率不再只是备注，应进入评分体系。
- 拥挤票不要放进核心观察位，只能放备选位。

### 3. D2 的“冲高回落”要继续保留，但必须强调“回落后不破结构”

观察结果：
- 不是只要冲高回落就行。
- 关键是回落后有没有守住 D1 支撑，收盘是否仍然健康。
- D2 是筛掉假强票的关键门槛。

建议：
- 冲高回落 + 不破支撑：保留
- 冲高回落但高开低走、收盘破位：剔除或降权
- D2 量比 2–3 倍的票，要有更强补强条件

### 4. D3 只做最终确认，不要承担“救票”功能

观察结果：
- D3 表现好的票，往往前面已经具备较好的 D1/D2 基础。
- D3 的角色是确认能不能继续走，不是把前面不好的票救回来。

建议：
- D3 核心关注是否回收观察价、是否站稳、是否承接。
- 不要因为 D3 盘中一度活跃，就把前面结构差的票抬高。

### 5. 选股池要分层，而不是一锅端

建议分层：
- 核心观察
- 强势确认观察
- 低频备选
- 剔除

目的：
- 核心池更干净
- 微信提醒更少噪音
- 后续验证也更容易看出哪类票真正有效

## 建议的策略修订方向

### D1 过滤规则

应补强：
- 炸板次数
- 首板封板干净度
- 成交额/换手的拥挤惩罚

应弱化：
- 只看是否首板封板成功
- 只看表面强度

### D2 过滤规则

应补强：
- 冲高回落后是否守住 D1 支撑
- 收盘是否接近平衡/十字
- 是否存在明显高开低走/出货特征

应保留：
- 量比温和或略高的优先权

### D3 观察规则

应补强：
- 是否回收观察价
- 是否真正站稳
- 是否有持续承接

应避免：
- 把 D3 当作救票层
- 只凭盘中一时活跃就提升优先级
